# Two-slots Mikrobus hat for Raspberry Pi

This hat enables the connection of two Mikrobus click boards to the RPI GPIO header.

## Tested
* [x] [MIKROE-4456 GNSS RTK Click](https://www.mikroe.com/gnss-rtk-click)
